# ChromBPNet / TransChromBP 学习资料说明

`docs/learning/` 现在不是“旧教程 + 几份补丁”，而是一套分层导航系统。

它的目标有四个：

1. 先把深度学习基础概念补齐，避免一上来被 `QKV / JSD / KL / loss` 卡住。
2. 让你看懂当前默认模型，而不是停留在旧 ChromBPNet 教学材料。
3. 让你知道项目到底做过哪些实验、每条线改了什么、结果怎样。
4. 让你分清“不接 foundation 的主线”和“接 foundation 的支线”。

## 3. 现在最容易搞错的三件事

### 3.1 旧 ChromBPNet 教学材料不等于当前默认实现

旧材料很有用，但主要负责补基础，不负责定义今天的默认主线。

### 3.2 foundation 线不等于当前主线

当前仓库里最稳定、最能直接拿来讲结果的，仍然是“不接 foundation”的 corrected-B / center-pool 主线。

### 3.3 很多实验已经有 verdict，不是都处在“继续试试”阶段

这一点推荐直接看：

- [TransChromBP项目实验全景图.md](TransChromBP项目实验全景图.md)
