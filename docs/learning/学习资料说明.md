## 4. 当前最该记住的 6 件事

1. 当前默认主线是 `TransChromBP teacher v2 + center pool`，不是旧教程里的 ChromBPNet。
2. Transformer 的收益成立，但当前不再写成“Transformer 特有 shortcut”。
3. `full/debiased gap` 是必要诊断，不是附加分析。
4. `debiased_profile_weight=2.0` 是当前最关键的训练语义之一。
5. `L3 shared-region` 是当前最强 external evidence。
6. no-bias 只是 supplementary boundary result，不改主线。

## 5. 现在不该再怎么用这个目录

不要再这样用：

- 从 `ChromBPNet完整项目深度解析.md` 直接开始，以为那就是当前实现
- 只看旧教程，不看当前模型文档
- 只看当前模型文档，不看内部复盘，于是搞不清为什么结论会变
- 把 no-bias 或旧 shortcut 叙事当成今天的主结论

更稳的方式是：

- **想补基础**：从基础层进入
- **想看当前实现**：从当前模型层进入
- **想理清项目与论文演化**：从内部复盘层进入

## 6. 一句话总结

今天的 `docs/learning/` 已经不是单一教程目录，而是一个分层导航系统：
