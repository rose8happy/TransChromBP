1. 让你阶段性回看代码时，知道先看哪里、为什么看。

## 7. 如果你只想快速判断“现在不该再怎么想”

不要再默认下面这些旧想法：

- “Transformer 特有 shortcut” 是当前主结论
- “纯卷积不会出现 shortcut”
- “strict compare 的核心还是 L2”
- “no-bias 已经证明 bias branch 绝对必要”

当前更稳的理解是：

- Transformer 的收益成立
- bias-safe 叙事比 shortcut 叙事更稳
- `L3 shared-region` 是当前最强 external evidence
- no-bias 只是 supplementary boundary result
