## 💡 关键公式和计算

### 感受野计算
```
第i层的感受野 ≈ 1 + 2 × (2^1 + 2^2 + ... + 2^i)
例如: 第9层 ≈ 1 + 2 × (2 + 4 + 8 + ... + 512) ≈ 2045 bp
```
📍 详见：BPNet模型小白教程.md "感受野是如何增长的"

### 卷积输出长度
```
output_length = input_length - kernel_size + 1
例如: 2114 - 21 + 1 = 2094
```
📍 详见：BPNet模型小白教程.md "计算长度变化"

### 膨胀卷积感受野
```
receptive_field = 1 + 2 × dilation_rate
例如: dilation=4 → RF = 1 + 2×4 = 9
```
📍 详见：BPNet模型小白教程.md "膨胀卷积详解"
