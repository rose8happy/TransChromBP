## 💡 核心要点预览

### BPNet是什么？
一个深度学习模型，用于预测DNA序列的染色质可及性。

### 三大创新：
1. **膨胀卷积** - 高效扩大感受野（9层看2000+ bp）
2. **残差连接** - 保留多尺度特征
3. **双输出头** - 同时预测分布(Profile)和总量(Counts)

### 模型架构：
```
输入: DNA序列 (2114 bp, one-hot编码)
  ↓
第一层卷积 (学习基本motifs)
  ↓
膨胀卷积 × 9层 (感受野指数增长)
  ↓
分支成两个头
  ↓
Profile预测 + Counts预测
```

## 🔧 快速参考

### 模型参数
```python
filters = 64               # 卷积核数量
n_dil_layers = 9           # 膨胀卷积层数
sequence_len = 2114        # 输入长度
out_pred_len = 1000        # 输出长度
counts_loss_weight = 5.0   # Counts损失权重
```

### 输入输出
```python
# 输入
Input: (batch_size, 2114, 4)  # DNA序列 one-hot

# 输出
Profile: (batch_size, 1000)   # 每个位置的信号
Counts: (batch_size, 1)       # 总信号强度
```
